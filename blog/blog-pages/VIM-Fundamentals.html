Vim is a text editor that is a clone of the vi editor, which was created around 1976. Despite its age, vim is one of the most popular editors in the world, and it is available on virtually every computing platform (or vi is available). vim has a bit of a learning curve, meaning that you may get somewhat frustrated with it until you have used it a few times.

Vim comes in many flavors :

  • NeoVim
  • spacevim
  • onivim2

The Terminology

  • Buffer : A buffer contains the text of the file and is what you edit. :h buffer
  • Window : Contains a buffer to display. Windows can be closed but the underlying buffer can remain in memory. :h window
  • Tabs : A tab is like another viewport. You can have many windows|splits open per tab. :h tab
  • Splits : A split simply refers to splitting the view port in N sections (various sizing and orientations available) to display windows. :h split
  • Motion : A command that moves the cursor (taken straight from the help docs, :h motion).

Opening Vim

Vim MODES

  • Normal: command mode esc
  • Insert: typing mode i
  • Visual v
  • Visual Line V

Basic navigation :

  • k, j, k, l : basic navigation
  • zz : scroll down
  • w : jump to next word
  • b : jump to previous word

Delete, Yank, Post & Visual Mode

  • yy : Copy line

  • p : Paste Line

  • v : Visual mode to select

    • y : copy
    • x : delete

  • :reg : registry of changes on the file

Insert Mode

  • i : left side of cursor
  • a : right side of cursor
  • I : left end of line
  • A : right end of line
  • o : insert new line below
  • O : insert new line above

Customization

  • :set scrolloff=8 : automatically scroll for me when reaching 8 last lines
  • :set number : show line numbers
  • :set relativenumber : set line numbers as relative to where the cursor is currently at

Saving your setting with .vimrc

  • :echo stdpath("config") : check when the .vimrc file is at

Other examples for .vimrc

set tabstop=4 softtabstop=4
set shiftwidth=4
set expandtab
set smartindent

Color Scheme

To select a color scheme just type:

  • :colorscheme <tab>

File & Navigation

Directory Navigation and Screen Splitting

  • $ vim . : opening vim on a directory

  • :Ex : Explore the current directory tree again, when u re inside a file

  • :Vex : (V)ertical (ex)plore

  • :Sex : (S)plit (ex)plore

  • CTRL-W :

    • v : split current windows vertically
    • s : split current windows horizontally
    • j : go to the downward window
    • k : go to the upper window
    • h : go to the window on the left
    • l : go to the window on the right
    • o : Close everything but the selected window
  • :e <filename> : edit a file in the current directory

Marks

you can bookmark a line in a file and then be able to jump into it from either the same file if it s a local mark marked with a lowercase letter, or from any other file if the mark is global with an uppercase letter mark.

  • m <lowercase_letter> : mark the current line as a local mark
  • m <UPPERCASE_LETTER> : mark the current line as a global mark
  • ' <letter> : Access the bookmarked line by typing the set letter of it

Alternate files & jumplist

  • CTRL-^ : equivalent to alt+tab on windows

Search & replace

  • /<the-word-to-search> : search for a word. Press n and N to go to next or previous

  • :set hls ic : add highlighting to all the found results

    • :nohls : to disable it

  • Select with v, then type : and then run the search by adding /<the-word-to-search>, this way you would run a search only in the selected region

  • :s/<word_to_replace>/<new_word> : replace selected word

  • :%s/<word_to_replace>/<new_word> : replace selected word over the whole file

  • :s/<word_to_replace>/<new_word>/g : replace selected word globally across that line

  • :s/<word_to_replace>/<new_word>/gc : replace selected word globally across that line, but ask for confirmation for each selected item

  • you can also use reg expression with () to select expressions into variables called \1 \2 \3 ...
